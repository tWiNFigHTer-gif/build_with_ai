project Name
LearnSync — Academic OS for High-Performers

Problem Statement
Students often study without timely guidance or structure, while teachers lack real-time visibility into where learners struggle—causing learning gaps to persist.

Project Description
LearnSync combines Socratic, no-spoiler assistance, Monk Mode focus timers, and an AI-managed to-do system to help students study effectively. For educators, it provides “Red Zone” friction heatmaps to pinpoint where students get stuck and enable faster, targeted intervention.

How Google AI Was Used
Socratic hint generation: AI produces guided hints and scaffolding without giving away answers.
Insight extraction for teachers: AI summarizes and groups common friction points into clear, actionable classroom signals.
Planning and prioritization: AI turns assignments and goals into structured, prioritized tasks for the to-do system
